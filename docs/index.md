# معرفی پروژه LinuxMirrors ایران

این پروژه با هدف تسهیل دسترسی کاربران ایرانی به مخازن نرم‌افزاری توزیع‌های لینوکس و ابزارهای کانتینر (مانند داکر) راه‌اندازی شده است. با توجه به محدودیت‌های اینترنتی و تحریم‌های ظالمانه، استفاده از میرورهای داخلی و بهینه امری ضروری است.

## ویژگی‌ها
- **سرعت بالا:** استفاده از سرورهای داخل ایران.
- **رفع محدودیت:** دور زدن تحریم‌های Docker Hub و مخازن رسمی.
- **نصب آسان:** اسکریپت‌های یک‌خطی برای تمامی توزیع‌ها.
- **پشتیبانی گسترده:** اوبونتو، دبیان، سنت‌او‌اس، فدورا، آرچ لینوکس و غیره.

## شروع سریع
برای نصب داکر با تنظیمات بهینه ایرانی:
```bash
curl -fsSL https://raw.githubusercontent.com/movtigroup/docker/main/install.sh | sudo bash
```
