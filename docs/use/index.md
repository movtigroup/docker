# راهنمای استفاده

برای استفاده از اسکریپت‌های این پروژه، می‌توانید از دستورات زیر استفاده کنید.

## نصب داکر (Docker)
این اسکریپت داکر را نصب کرده و میرورهای ایرانی را به صورت خودکار تنظیم می‌کند:
```bash
curl -fsSL https://raw.githubusercontent.com/movtigroup/docker/main/install.sh | sudo bash
```

## تنظیم میرور داکر (برای داکر نصب شده)
اگر داکر را قبلاً نصب کرده‌اید و فقط نیاز به تنظیم میرور دارید:
```bash
curl -fsSL https://raw.githubusercontent.com/movtigroup/docker/main/mirror.sh | sudo bash
```

## تغییر مخازن سیستم‌عامل (به زودی)
اسکریپت جامع تغییر سورس‌های سیستم‌عامل در حال توسعه است.
