# ابزارهای کاربردی دیگر

در این بخش ابزارهایی که می‌توانند به شما در مدیریت بهتر سیستم‌های لینوکسی و کانتینرها کمک کنند معرفی می‌شوند.

## دی‌ان‌اس‌های پیشنهادی (DNS)
برای بهبود سرعت و رفع برخی محدودیت‌ها، استفاده از دی‌ان‌اس‌های ایرانی پیشنهاد می‌شود:
- **شکن (Shecan):** `178.22.122.100`
- **بگذر (Begozar):** `185.51.200.2`
- **رادار گیم (Radar Game):** `10.202.10.10`

## مدیریت کانتینر
- **Portainer:** ابزاری گرافیکی برای مدیریت داکر.
- **LazyDocker:** ابزاری ترمینالی (TUI) برای مدیریت کانتینرها.
